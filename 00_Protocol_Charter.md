
# i👁️ — Neutral Protocol Charter (Attribution-Free)

## Core
- Fixed supply: **12,000,000,000,000,000,000**
- Settlement: deterministic (chain rules)
- Accounting: transparent/observable (on-chain events)
- By-design neutrality: no persona, no narrative, no promises

## Non-claims
- Not an investment promise
- Not a yield guarantee
- Not a passive-income commitment

## “Translucent accounting”
Meaning: the system emits sufficient public events to reconstruct flows, while preserving wallet-level privacy as allowed by the chain.
For EVM: standard Transfer events + optional PaymentSplit events.
